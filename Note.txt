‚úÖ 1. User Authentication (Login/Register)

    * Routes: /login, /register, /logout
    * Models: User
    * Use bcrypt for password hashing
    * Session/cookie-based login

‚úÖ 2. Accounts Module

    * Purpose: Manage user's bank accounts
    * Features: Add/Edit/Delete bank accounts
    * DB Table: accounts (id, user_id, bank_name, balance, type)

‚úÖ 3. Transactions Module

    * Purpose: Record income & expenses
    * Features: CRUD + Filter by category/date
    * DB Table: transactions (id, user_id, account_id, amount, type, category, date, note)

‚úÖ 4. Budgeting Module

    * Purpose: Set monthly budget by category
    * DB Table: budgets (id, user_id, category, amount, month)

‚úÖ 5. Investments Module

    * Purpose: Track investments (stocks, crypto, FD, etc.)
    * DB Table: investments (id, user_id, type, name, value, date)

‚úÖ 6. Goals Module

    * Purpose: Financial goal setting (e.g., save ‚Çπ1 lakh)
    * DB Table: goals (id, user_id, goal_name, target_amount, current_amount, deadline)

‚úÖ 7. Reports Module

    * Purpose: Monthly/yearly summary, charts
    * Features: Use Chart.js to visualize data
    * Backend: Aggregate data using Sequelize queries

‚úÖ 8. Bills & Subscriptions

    * Purpose: Track recurring payments
    * DB Table: subscriptions (id, user_id, name, amount, frequency, next_due_date)

‚úÖ 9. Settings + Help

    * Settings: Update profile, change password
    * Help: Static page or FAQs
